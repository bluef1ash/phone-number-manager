# 社区居民联系电话管理系统

此系统使用Spring Boot + Mybatis + Spring Security + Node.js + Webpack + Vue.js架构编写。数据库采用MySQL，提供**强大的**、**安全的**和**完整的**管理社区的功能。

## 安全性

此系统在系统层面提供了众多的安全特性，产品安全无忧。这些特性包括：

- Session会话验证
- 系统用户“单点登录”
- XSS安全防护
- 表单自动验证
- 输入数据过滤
- 表单令牌验证
