# 社区居民联系电话管理系统

此系统使用Spring + Mybatis + Spring Security Spring MVC + Webpack + Vue.js架构编写。数据库采用MySQL，提供强大的、安全的和完整的管理社区的功能。

## 安全性

此系统在系统层面提供了众多的安全特性，产品安全无忧。这些特性包括：

- Session会话验证
- 系统用户“单点登录”
- XSS安全防护
- 表单自动验证
- 输入数据过滤
- 表单令牌验证
